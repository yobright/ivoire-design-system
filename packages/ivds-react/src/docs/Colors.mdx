# Couleurs

Le systeme de couleurs IVDS est organise en trois couches : **brand**, **semantic** et **UI**.

---

## Brand — Identite visuelle

Les couleurs brand incarnent l'identite ivoirienne. Le **burnt orange** primaire
evoque la terre et la chaleur du pays, tandis que le **vert institutionnel** represente
la croissance et le developpement.

### Primary — Burnt Orange

|-------|--------|-------|
| `--color-brand-primary-50` | `#fef5ee` | Backgrounds subtils |

| `--color-brand-primary-400` | `#e8853a` | Actions secondaires |

## Semantic — Feedback utilisateur

Les couleurs semantiques communiquent l'etat du systeme a l'utilisateur.

| Categorie | Token 500 | Usage |
|-----------|-----------|-------|
| Success | `#19895d` | Validation, confirmation |
| Warning | `#ce790b` | Avertissement, attention |
| Error | `#d73a50` | Erreur, danger |
| Info | `#1f6fe5` | Information, aide |

---

## Usage en CSS

```css
.my-component {
  background-color: var(--color-brand-primary-50);
  color: var(--color-brand-primary-900);
  border: 1px solid var(--color-brand-primary-200);
}

.my-component:hover {
  background-color: var(--color-brand-primary-100);
}
```

## Usage en React

```tsx
import { ThemeProvider } from '@ivds/react';

// Les tokens sont disponibles via CSS custom properties
<ThemeProvider mode="light">
  <Button variant="primary">Action</Button>
</ThemeProvider>
```
