# Politique de securite

## Versions supportees

| Version | Support |
|---------|---------|
| 1.x | Correctifs de securite actifs |
| < 1.0 | Non supporte |

## Signaler une vulnerabilite

Si vous decouvrez une faille de securite dans IVDS :

1. **Ne creez pas d'issue publique**
2. Ouvrez un [rapport de securite prive](https://github.com/babiverse/ivoire-design-system/security/advisories/new) sur GitHub avec :
   - Description de la vulnerabilite
   - Etapes pour reproduire
   - Impact potentiel
3. Vous recevrez un accuse de reception sous **72 heures**
4. Le probleme sera corrige avant toute divulgation publique

Merci de contribuer a la securite d'IVDS.
